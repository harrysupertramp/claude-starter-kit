# Learnings

- The ~40% noise estimate is accurate. Most extracted pairs are greetings, "ok", tool approvals, or normal Q&A. Skip quickly and focus on corrections/preferences.
- Skill-specific LEARNINGS.md files are valid routing targets. When a learning is about a specific skill's behavior, route it to that skill's LEARNINGS.md.
- Always read the target file before editing — it may have changed since the routing table was written.
- When presenting findings in Phase 3, group by target file — easier for the user to approve/reject when related changes are together.
